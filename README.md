# Portfolio Professionnel - Mathys Dauvergne

Bienvenue sur le dépôt de mon portfolio, réalisé dans le cadre de mon **BTS SIO (Services Informatiques aux Organisations)**, option **SLAM** (Solutions Logicielles et Applications Métier).

Ce projet présente mon parcours académique, mes compétences techniques, ainsi que les projets et stages réalisés durant ma formation.

## Technologies & Outils

### Développement (SLAM)
* **Langages :** HTML5, CSS3, JavaScript, PHP (PDO & MVC), C# (.NET), Python, Lua.
* **Frameworks & Bibliothèques :** Intégration d'API, manipulation du DOM.
* **Outils :** Visual Studio, PhpStorm, GitHub (Versionnage).

### Infrastructure & Systèmes
* **Bases de données :** MySQL, SQL Server (Conception et requêtes).
* **Systèmes :** Linux (Debian, Ubuntu), Windows Server (AD/DNS/DHCP).
* **Virtualisation & Réseau :** VMware, VirtualBox, CISCO Packet Tracer.

## Fonctionnalités du Portfolio

* **Mode Sombre / Mode Clair :** Interface dynamique avec persistance du choix via `localStorage`.
* **Design Responsive :** Adapté à tous les supports (Mobile, Tablette, Desktop).
* **Compétences Visuelles :** Présentation épurée par catégories (Développement vs Infrastructure) avec logos officiels.
* **Section Stages :** Détails des missions effectuées chez *leMAGASINduBTP.com* et *Bois Gestion*.
